## 1. 프로젝트 개요

### 1.1 프로젝트 목표

> **"Excel 기반 IT 자산관리를 웹 시스템으로 완전히 대체하여,
> 등록/수정/조회/삭제 및 문서 출력까지 가능한 통합 관리 시스템 구축"**

### 1.2 현재 상황 (As-Is)

```
[현재 업무 방식]

전산팀 담당자
    │
    ├── 장비 입고 → Excel 열기 → 02IP-MAC관리.xlsx에 등록
    │
    ├── IP 할당 → Excel 열기 → 해당 시트 찾기 → 수동 입력
    │
    ├── 토너 교체 → Excel 열기 → 01프린트관리대장.xlsx → 해당 월 컬럼에 날짜 입력
    │
    ├── IT 요청 접수 → Excel 열기 → 06전산업무의뢰서.xlsx에 추가
    │
    └── 현황 보고 → Excel 데이터 수작업 집계 → 보고서 작성
```

**문제점:**
- 5개 Excel 파일, 70개 이상의 시트 관리
- 데이터 중복 및 불일치
- 검색/집계에 많은 시간 소요
- 여러 사람이 동시 수정 시 충돌
- 보고서 작성 시 수작업 필요

### 1.3 목표 시스템 (To-Be)

```
[목표 업무 방식]

전산팀 담당자
    │
    ├── 장비 입고 → 웹 접속 → [장비등록] 버튼 → 폼 입력 → 저장
    │
    ├── IP 할당 → 웹에서 장비 선택 → IP 자동 할당 또는 수동 지정
    │
    ├── 토너 교체 → 웹에서 프린터 선택 → [토너교체] 버튼 → 자동 이력 기록
    │
    ├── IT 요청 접수 → 웹에서 [새 요청] → 진행상태 실시간 관리
    │
    └── 현황 보고 → 대시보드에서 즉시 확인 → [Excel 다운로드] 또는 [보고서 출력]
```

### 2.2 기능별 상세 설명

#### 2.2.1 장비 관리

| 기능 | 설명 | 화면 예시 |
|------|------|----------|
| 장비 등록 | PC/노트북/서버/프린터/네트워크 장비 신규 등록 | 폼 입력 → 저장 |
| 장비 수정 | 기존 장비 정보 수정 | 목록에서 선택 → 수정 |
| 장비 조회 | 목록 조회, 상세 조회, 검색/필터 | 테이블 + 필터 |
| 장비 폐기 | 폐기 처리 (이력 보존) | 상태 변경 |
| 장비 할당 | 사용자에게 장비 할당/회수 | 사용자 선택 → 할당 |
| 할당 이력 | 누가 언제부터 언제까지 사용했는지 | 타임라인 뷰 |

**등록 시 입력 항목:**
```
- 관리번호 (자동채번 또는 수동)
- 장비구분 (PC/노트북/서버/프린터/네트워크/기타)
- 세부구분 (L2스위치, L3스위치, NAS 등)
- 모델명
- S/N (시리얼번호)
- 구매일
- 설치위치 (1층/2층/3층/2공장)
- 사용부서
- OS, CPU, 메모리, 저장장치 (PC/서버인 경우)
- 비고
```

## 4. 데이터베이스 설계

### 4.1 테이블 구성 (15개)

```
[마스터 테이블]
├── equipment         : 장비 정보
├── users             : 사용자 정보
├── software_license  : SW 라이선스
└── printer_consumable: 프린터 소모품

[관계/이력 테이블]
├── equipment_user    : 장비-사용자 할당 이력
├── ip_allocation     : IP 할당
├── sw_installation   : SW 설치 현황
└── consumable_history: 토너 교체 이력

[인증정보 테이블] (보안 분리)
├── user_credential      : PC/메일 비밀번호
└── equipment_credential : BIOS/BitLocker 비밀번호

[업무 테이블]
├── it_request     : IT 요청
├── payment        : 품의/지출
└── regular_payment: 정기지불

[네트워크 테이블]
├── network_policy : 방화벽 정책
└── wireless_ap    : 무선 AP
```

### 4.2 주요 테이블 관계

```
                    ┌─────────────────┐
                    │     users       │
                    │    (사용자)      │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
    ┌─────────────────┐ ┌─────────┐ ┌─────────────────┐
    │ user_credential │ │equipment│ │ sw_installation │
    │   (인증정보)     │ │ _user   │ │   (SW설치)      │

                        └────┬────┘
                             │
                    ┌────────┴────────┐
                    │    equipment    │
```
